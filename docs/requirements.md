# 开发需求文档

## 项目背景
情侣周年纪念网站，记录从 2023 年 6 月 11 日到 2026 年 6 月 11 日的回忆。

## 用户画像
- 两位用户共用同一账号
- 用户在国内，不翻墙，需要国内可直接访问的免费方案
- 访问设备：电脑、手机、平板

## 功能需求

### FR-0 开场页
- 打开网站显示 "Loading Memories" 加载进度条（2.5 秒）
- 加载完成后显示 Enter 按钮
- 点击 Enter：已登录 → 主页，未登录 → 登录页

### FR-1 登录/注册
- 邮箱 + 密码登录
- 邮箱 + 密码注册
- 两人共用同一账号

### FR-2 主页（双入口）
- 左侧：艺术字 "TIME" + 信封造型 → 点击进入时间轴页
- 右侧：艺术字 "MAP" + 3D 地球 → 点击进入城市地图页
- 手机端上下堆叠

### FR-3 时间轴页
- 日期选择器（只列出有日志或照片的日期）
- 默认显示最早有记录日期的日志和照片
- 左侧：日志列表（展示、新增、编辑、删除）
- 右侧：照片网格（展示、上传、删除）
- 日志为纯文字（标题 + 内容）
- 日志和照片按日期关联
- 切换日期后左右内容联动更新
- 新增日志/照片时需指定日期

### FR-4 城市地图页
- 显示中国 SVG 地图（覆盖 6 省份约 71 个城市）
- 有照片的城市 → 暖色高亮（已激活）
- 无照片的城市 → 浅灰半透明（未激活）
- 点击城市 → 弹出照片画廊浮层
- 照片画廊：查看、上传（自动关联城市 + 选择日期）、删除
- 删除城市最后一张照片 → 变为未激活
- 点击空白处取消选中城市

### FR-5 数据范围
- 时间范围：2023.06.11 — 2026.06.11
- 城市范围：广东省、福建省、江苏省、浙江省、上海市、安徽省 各地级市

## 非功能需求

### NFR-1 响应式
- 桌面（≥1024px）：左右分栏
- 平板（768-1024px）：上下堆叠，适量缩小
- 手机（<768px）：上下堆叠，全屏适配

### NFR-2 国内可访问
- 不使用被墙服务（Google/Firebase）
- 使用 Supabase + GitHub Pages

### NFR-3 免费
- 所有服务使用免费额度
- 图片上传前端压缩 ≤ 2MB

### NFR-4 简洁直观
- 暖粉/珊瑚色主色调
- 艺术字入口
- 流畅动效
